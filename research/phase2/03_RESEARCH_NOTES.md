# Research Notes — Phase 2

Existing observations regrouped by research theme rather than by website. Evidence only — nothing merged, generalized, compared, or interpreted. Each observation is reproduced under the website it came from.

Website order within each theme follows the source research order: Linear, Stripe, Vercel, Brittany Chiang, Paco Coursey, Lee Robinson, Josh W. Comeau, Raycast, Notion, Resend, shadcn/ui, Primer / GitHub, Shopify Polaris, Atlassian Design, Tailwind CSS.

No "Common Patterns" section is included: producing one would require merging evidence into generalized statements, which this file excludes by rule.

- Hero → Why Tailwind → How it works → Tailwind in the wild → Ready-made components (Plus upsell) → footer. Top nav: Docs, Blog, Showcase, Partners, Plus, GitHub. Footer grouped: CSS, Resources, Plus, Community.
